# Bima+ Package Injector (FREE)

Sebuah aplikasi sederhana dan bebas yang dapat digunakan untuk mendaftarkan paket data kartu tri.

# Requirement

1. NodeJS>=16

# Installation

1. `git clone https://github.com/vermaysha/bima-plus.git`
2. `npm install`
3. `npm run build`
4. `npm run start`
